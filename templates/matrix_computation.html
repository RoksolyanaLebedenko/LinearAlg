Solving Ax = b via LU-factorization

If A = LU, then Ax = LUx = b . Thus, first solve Ly = b, for y, then solve Ux = y for x. Both solutions are found by back-substitution.

Useful information

LU-factorization factors a matrix as the product of a lower triangular matrix and an upper triangular matrix. The LU-factorization can be viewed as the matrix form of Gaussian elimination.
